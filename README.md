# ArenaOfValor
傳說對決 - update by 2018-05-16

# 前言

跟福爺組隊大約 10場左右吧，發現到臨時加入的強手，真的要一起打排位時，會有不協調的問題，被敵方打崩！

這里有想過有可能是5排的關係，對手也跟著5排，強度提高了，但我方觀念沒有整合好。

這里把戰隊大方向的戰略，先整理一下，不要自已玩自已的，減少個人英雄主義，多尊重隊友。

# 排位戰略

### 第1話：4級前，團隊：要報敵方人數，避免被 GANK。

01. 要反野，一定是兩人以上去對方的野區，一人單走，很容易送頭，而實際上在排位時，依個人經驗，就算搶到紅、藍BUFF，也不見得贏得比賽，風險很高。

02. 首先，打野不能死，邊線角，要幫忙顧紅、顧藍，中路角看到對方反野，馬上移動野區開戰。

03. 個人經驗，這個時期，中路是主角，而不是打野，中路快速移動多放一、兩招，往往能增加碰撞的勝算，打野畢竟這時要清野，輔助的工作是視野居多。

04. 4級前，強勢的角色以1技、2技能強控的角色有巨大優勢，往往2名刺客配合好能集中秒軟皮角，如：蝙蝠俠、呂布、薇拉，呂布、凡恩、薩尼、貂蟬、哥德爾、渥馬爾、愛里、拉茲、超人、龍馬、閃電俠、洛克。以上的角色，4級前是 T1角。


### 第2話：4級後，團隊：要報大招是否有，讓打野能收割。

01. 通常角色的大招好了，會有綠燈，但克萊斯需要怒氣值，這個要自已主動回報。

02. 4級前，中路、魔龍路，看魔龍出生了，要盡量比敵方早5秒去蹲草，為優。

03. 打野，在魔龍出生時，有兩個選擇，一是骨龍，二是魔龍，打野選骨龍，凱撒位人員要到，打野選魔龍，魔龍位人員要到。

04. 各路看到打野靠近，可吃招，用少血，吸引敵方過來，讓打野出手，就收頭。

05. 看到敵方打野的所在位置，馬上判斷要移動或開魔龍。

### 第3話：正確的觀念

01. 傳說對決，要打排位要用 T1角來玩，不用就是掉塔、送頭。

02. 通常有好好守塔，掉塔很難發生，但傳說對決流行的是吃定軟皮角，塔下殺人，所以自已是軟皮角的，請多自愛，要多撐2秒，我方才能回守，不要輕易出塔，兵線優先處理。

03. 這版本，玩的是兩個以上位移角，在 6級至10級時，盡量配合連續技塔下殺人、清野，如果我方沒有位移角，會連對方的後排角，都無法碰到。然後就會贏。

04. 雙射手，遇刀鋒、閃電俠，必不斷送頭，無法保排。

05. 兵線是第一，會殺人，不清兵，不適合打排拉。



### 第4話：戰隊目前的組合
以上只先分析角色會遇到的場景，不針對個人。


#### 魔龍路(小丑)

  01. 要能有對線的抗力，讓輔助能離開，輔助上4級後，一定要去參加打野的團戰，而輔助一離開，魔龍路就會被敵方集結打塔。

  02. 小丑在塔下的自保能力高，實戰中敵方會採取 3隻實施塔下強殺。此時我方要比的，是中野輔集合的速度，目前本團很缺乏中野輔聯動的能力，遇敵方3人衝塔，就爆塔。

  03. 小丑怕的是魔傷角，如圖倫、閃電俠、莉莉安，配合刺客角，在小丑未察覺的情況，常常爆發擊殺。

  04. 小丑，能單吃軟皮角。
  
  05. 小丑，對上坦克，消耗敵方血量就好，硬要殺坦克，時間會超過10秒，自主判斷該不該閃，通常3秒鐘的空檔，敵方就過來了。
  
  06. 小丑，有高機動性，成熟的小丑，會用右手姆指，移動鏡頭，在擁有視野的情形下，找好位置攻擊，讓敵方陷入被動。
  

#### 魔龍路(琳蒂)

  01. 裝備沒有2件前，很難殺人，要靠近草，多用被動打對方，也要時常看小地圖，避免被對方刺客針對，最近連血線，也盡量保持一半以上。
  
  02. 沒魔請先買回 `清醒吊墜 $120`, 減少回家次數，能多賺兩波兵線。沒錢就沒裝，射手會拖累整個隊伍。
  
  03. 開魔龍請報備，因為血少，讓大家好保排。
  
  04. 開凱撒請報備，因為血少，讓大家好保排
  
  05. 露頭就是死，假設一場比賽是20分鐘，死亡次數少於 2次，才是A排級的射手。
  
  06. 露頭就要換位置，自已要準備被人 GANK。
  
  07. 當我方掉很多塔時，射手要負責清兩條以上的兵線，因為最安全也賺最多錢，買裝。
  
  08. 當我方掉很多塔時，輔助還要清兵線，沒有餘力去作視野，這個缺失射手要負責50%。
 
  09. 琳蒂跟打野角兩人配合時，適合攻擊性的戰術，如：偷野、小型碰撞。
  
  10. 琳蒂，跟勇、特爾不同，不適合龜塔下發育，前、中期在野區、河道上，隊友能做視野，就是琳蒂發育的好時機。
  

#### 凱撒路(馬洛斯)

  01. 遊走型的馬洛斯，主要是清兵，這不強，主因：清兵原本就是各路自已該負責的。

  02. 遊走型的馬洛斯，通常配 `遊蕩者之靴 $630` ，若對方法師是強控，一定要換 `聖騎士戰𦖹 $690`，馬洛期被控制1秒以上，是會被秒的。

  03. 大絕：請多注意，您的大絕，一定要針對輸出位、軟皮角，能震幾個，就算幾個。

  04. 大絕：如果跳大絕後，沒有隊友集火，有可能是您沒講好，或是您跳到是血多或位移角，一場比賽，發生兩次大絕的失誤，這個馬洛斯不強。

  05. 在凱撒路時，換血請小心處理，若對方被您換到半血左右，請回報打野收割。


### 中路(閃電俠)

  01. 不會用被動，一定一出場就半血，會不斷回家。
  
  02. 閃電俠是後期角，在6級前，以 1技+被動+2技，無損消耗敵方血為主要節奏，由其是塔下的射手，必抓。
  
  03. 閃電俠入侵野區前，事先要看小地圖，先把自已的目標定出來，如：我猜打野在清藍、清紅，我要1技+2技搶BUFF。遇到意外，像是有人埋伏或是遇到2支以上，一定要退場，因為敵方的中路法師，一定會實施3夾1。
  
  04. 閃電俠的主攻是2技，2技的出招範圍很小，很要求命中度，2技的命中提高了，才能多多收割。
  
  05. 閃電俠，有最強的位移技，故支援各路的工作非常重要，一清完兵，就要想清楚各路要支援的點在那。


#### 輔助(贊尼爾)

  01. 能守，不能殺，逆風局時，無法帶風向
  
  02. 要帶兵線推塔。

  03. 不能殺人，不是贊尼爾的問題，贊尼爾，只能飛下來，然後放放技能，就要退場了，無法做收割，坦度不是很高。
  
  04. 贊尼爾遇逆風局，只能清小兵，敵方多買1件裝備，贊尼爾就廢了。
  
  05. 贊尼爾不是保排角。
  
  06. 贊尼爾是分推、帶線角。
 
#### 輔助(克萊斯)

 01. 大絕要怒氣，所以要扣著，推塔用，效益高。
 
 02. 克萊斯，野區時，有很多障礙物，敵方好躲好拉打，開大絕的效益，會減低。 
 
 03. 大絕要很久，盡量不要拿來救人，敵方看到您沒大，就呵呵。
 
 04. 2技，請跳軟皮角，最好是常駐。
 
 05. 1技，很強


### 打野(凱格路士)

  01. 2件裝以前，一開戰就易送頭，當凱格路士，在換血時，極需要隊友的協助。
  
  02. 裝滿神角
  
  03. 在團戰時，凱格路士，易被集火被秒，很依賴隊友先開戰或先開視野，不能保排。
  
  04. 很有可能在6分鐘前，沒有好的開戰時機，此時清野存錢，是最重要的，不開戰，不要緊。
  
  05. 2018-05-16, 凱格路士角色搭配上要求大男陣，讓凱格路士大絕一開，能撐完3秒鐘。
  
  06. 凱格路士，要4件裝備以後，才能開戰，在那之前，入侵野區：2隻集火能秒凱格路士，團戰：血少難撐。
  
  07. 凱格路士，發育時，敵方會不斷入侵野區，讓凱格路士無法發育，連塔也無法去守。
  
  08. 打野位上，凱格路士跟蝙蝠俠不同，在 6級前，蝙蝠俠能隱形開團，吃中路，吃魔龍射手。凱格路士沒裝，無法發揮。
  
  09. 打野位上，凱格路士跟刀鋒不同，在 4級後，刀鋒只要抓進場時機，就能刷頭。凱格路士沒裝，沒坦。
  
  10. 我方組合，在前期，要有能抗住敵方中野輔衝塔的攻勢，讓凱格路士發育。
